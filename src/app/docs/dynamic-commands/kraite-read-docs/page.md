---
title: kraite-read-docs
---

Loads the functional map for the active Kraite project before implementation
or operations. {% .lead %}

The command starts from the shared documentation index, then selects the
relevant architecture, feature, admin, mobile, or deployment chapters. Live
server facts come from the credential registry, never from retired fleet notes.

For code behavior, documentation provides intent and the repository provides
implementation evidence. Neither source substitutes for the other.
